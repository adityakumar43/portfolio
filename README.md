# 🚀 Personal Portfolio Website

A modern, responsive, and visually appealing Personal Portfolio Website built using HTML, CSS, and JavaScript. This portfolio showcases my skills, projects, internship experience, and contact information in a professional manner.

## 📌 Overview

This portfolio website serves as my digital presence and professional profile. It highlights my technical skills, projects, educational background, and contact details while providing recruiters and visitors with an engaging user experience.

The website features a modern dark-themed UI with gradient backgrounds, glassmorphism effects, smooth animations, responsive layouts, and interactive elements.

---

## 🎯 Objectives

* Create a professional online portfolio.
* Showcase projects and technical skills.
* Highlight internship and academic achievements.
* Provide an easy way for recruiters to connect.
* Demonstrate frontend development skills.

## 💻 Deployment

### GitHub Pages

1. Push project to GitHub
2. Open Repository Settings
3. Navigate to Pages
4. Select branch
5. Save changes

Portfolio will be live at:

```text
https://yourusername.github.io/portfolio/
```

### Netlify

1. Login to Netlify
2. Import GitHub repository
3. Deploy site

### Vercel

1. Import repository
2. Deploy project

### AWS S3

1. Create S3 bucket
2. Enable Static Website Hosting
3. Upload project files
4. Configure bucket policy
5. Access website using S3 endpoint
